# ShaikGraphix - Premium Printing Calculator

A web-based sticker and artboard/artpaper printing price calculator.

## Files
- `index.html` - Main HTML file
- `styles.css` - Styling
- `script.js` - JavaScript logic

## Deployment Options

1. Go to https://www.netlify.com/
2. Sign up/Login
3. Drag and drop this folder onto Netlify dashboard
4. Your site is live!
